# robosys2025_kadai01
---

##　addwwwの説明

ターミナル上で実行できる入力された標準入力にwwwを足してくれるコマンドです。　
単語に柔らかい印象を与える事ができます。

---


## GitHub Actions による自動テスト

GitHub Actions で、  
以下の内容が自動チェックされます：

- 標準入力を処理できるか  
- 出力が正しいか  

github/workflows/test.yml で設定しています。

---

## 動作環境

- OS: Linux / WSL2 / macOS
- Python 3.x
- bash（/bin/bash）
- GitHub Actions: Ubuntu-latest
