##　addwwwの説明

ターミナル上で実行できる入力された標準入力の文章にwwwを足してくれるコマンドです。
標準出力した上でPDFで出力します。
単語に柔らかい印象を与える事ができます。
改行　空行に対応しました。


---

## 動作環境

- OS: Linux / WSL2 / macOS
- Python 3.x
- bassh）
- GitHub Actions: Ubuntu-latest


---

## 入力出力の例
　
例1：1行の文章（基本）

入力（標準入力）

echo "こんにちは" | ./addwww


標準出力

こんにちはｗｗｗ


生成されるPDF

output.pdf

内容：

こんにちはｗｗｗ

例2：複数行の文章

入力

printf "今日は良い天気\n明日も晴れる\n" | ./addwww


標準出力

今日は良い天気ｗｗｗ
明日も晴れるｗｗｗ


PDF内容

今日は良い天気ｗｗｗ
明日も晴れるｗｗｗ

例3：空行（空白行）を含む入力

入力

printf "一行目\n\n三行目\n" | ./addwww


標準出力

一行目ｗｗｗ

三行目ｗｗｗ


PDF内容

一行目ｗｗｗ

三行目ｗｗｗ


※ 空行は削除されず、そのまま保持されます。

例4：スペースを含む文章

入力

cat << EOF | ./addwww
これは テスト です
空白   が   多い 行

最後の行
EOF


標準出力

これは テスト ですｗｗｗ
空白   が   多い 行ｗｗｗ

最後の行ｗｗｗ


PDF内容

これは テスト ですｗｗｗ
空白   が   多い 行ｗｗｗ

最後の行ｗｗｗ

例5：ファイルを入力として使用

入力ファイル（input.txt）

ロボットシステム
工学


実行

cat input.txt | ./wwpdf


標準出力

ロボットシステムｗｗｗ
工学ｗｗｗ


PDF

output.pdf に同内容が出力される
